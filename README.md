## 依赖

- vite
- vue3 + vuex 4.x + vue-router 4.x
- typescript
- tsx(好像对指令的支持不太好，也可能是我没用明白)
- ant-design-vue
- echarts

## setup
```bash
# Modules download
npm i

# Compiles and hot-reloads for development
npm run dev

# Compiles and minifies for production
# 因为买的丐版服务器，所以使用本地打包
npm run build
```
