# 🚧 vite-typescript-antd

## 依赖

- vite
- vite-jsx(支持 vue 原生指令：v-if, v-model, v-show……)
- vue3 + vuex 4.x + vue-router 4.x
- typescript
- tsx
- ant-design-vue
- echarts

## setup
```bash
# Modules download
npm i

# Compiles and hot-reloads for development
npm run dev

# Compiles and minifies for production
npm run build
```
